There are several listings all over the Internet on all the problems associated with the 532h and the Ubuntu Netbook Edition, but I'd like to consolidate some of the problems.  This guide will hopefully add more redundancy to the searches made by others and point them in the right direction.

All solutions I've come across are from here https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks

Known problems:

  1. Trackpad multigesture support
  2. Trackpad side scrolling support
  3. SD Card Reader
  4. Microphone
  5. Some Fn + Key combinations
  6. Brightness controls are a bit funky

Some Solutions

  • Trackpad side scroller fix

    Go to terminal and type the command:

    gksu gedit /etc/modprobe.d/psmouse.conf

    When that comes up add the line:

    options psmouse proto=imps

    Reboot and it should work

    • Microphone workaround

    To fix the microphone is a bit more difficult, there is no permanent fix that I'm aware of

    You just need to in the terminal type the following:

    /bin/sh/ -c "PULSE-SERVER=127.0.0.1 program-name"

    Where program name is the command line name of skype, gnome-sound-recorder, etc.
